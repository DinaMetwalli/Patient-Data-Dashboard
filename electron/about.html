About Medical Dashboard

Welcome to the Medical Dashboard! This dashboard provides various features to manage and analyze patient data effectively.

Features:

  • View All Patient Data: Access comprehensive information about all registered patients.
  • Analyze Patient Data: Analyze patient data to identify trends and patterns.
  • Reports: Generate reports based on patient data analysis.
  • Patient Data: Manage individual patient records.

Technology Used:

The Medical Dashboard is built using HTML, CSS, and JavaScript. It utilizes Chart.js library for data visualization.
